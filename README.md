🚀 Getting Started
Prerequisites
Node.js installed
Create your own .env file (see .env.example if provided)
.env PORT=Define_port MONGO_URI=mongodb://localhost:27017/Your_Database_Name OLLAMA_BASE_URL=local_host_Ollama_URL OLLAMA_MODEL=llama3.2:latest OLLAMA_NUM_GPU=0 NODE_ENV=development
⚠️ node_modules and .env files are excluded from this repo.

🔧 Backend Setup
cd ./backend
npm install
npm start
💻 Frontend Setup
cd ./frontend
npm install
npm run dev
